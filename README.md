# demo-spring

> 预计组装成自己的starter

- spring-jpa

- spring-mail

- spring security

- jwt

- redis

- 目录结构
```
├─java
│  └─com
│      └─isxcode
│          └─demo
│              ├─annotation        - 自定义注解
│              ├─common            - 通用类
│              ├─config            - 配置中心
│              ├─exception         - 异常管理类
│              ├─filter            - 拦截器
│              ├─moudle 
│              │  └─properties     - 项目参数配置
│              └─utils             - 工具类
└─resources
    └─templates                    - 代码自动生成模板
```
