## 通过gradle发布jar到中央仓库

1- 注册账号   https://issues.sonatype.org/
2- 提交issue   Open Source Project Repository Hosting (OSSRH)
3- 写代码
4- 配置gradle
5- 命令上传到临时仓库
6- 发布到中央仓库
