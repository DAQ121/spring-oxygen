##### vim
- mode
esc - choose mode
v - visual mode

- move
h - left
j - down
k - up
l - right
o - new next line and edit
w - next
b - previous
e - end
:+ n - move to n line
zz - move to mid screen
ctrl+shift+j - move down code block
ctrl+shift+k - move up code block

- edit
dd - delete line
d - only delete 
O - new previous line and edit
s - delete and edit
i - insert left
I - insert line left
a - insert right
A - insert line right

- common
ctrl+a - choose all 
ctrl+c - copy
ctrl+x - cut 
ctrl+v - paste
ctrl+z - redo
ctrl+s - save
ctrl+f - find
ctrl+shift+f -  find int
